# 개관
 - 어떠한 알고리즘이 생겨난 배경과, 설계하는데 필요했던 결정적 통찰을 얻을 수 있을까?
 - 경험과 깨달음을 얻기 위한 좋은 통로가 바로 프로그래밍 대회
  - 알고리즘에 사용된 원칙들을 이해하고 변형해야 풀 수 있는 문제들이 많이 출제되기 때문
  - 공정하고 즉각적인 피드백이 주어지기에, 재미있게 공부할 수 있다.
